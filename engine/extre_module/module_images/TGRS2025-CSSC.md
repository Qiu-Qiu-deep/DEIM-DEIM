# CSSC

## 1. 动机

### 现有方法的问题
- **缺乏全局列特征建模**：现有的注意力机制和列校准模块未能有效建立列的长程依赖关系，无法在全局上下文中探索和调制特征列模式 [3]
- **列特征增强不足**：由于缺乏对长程依赖的显式建模，现有方法难以充分捕捉全局上下文中的列特征 [3]

### 提出模块的目的
设计CSSC（列空间自校正，Column Spatial Self-Correction）模块，通过整合三个关键信息来全面捕捉全局上下文中的列特征 [3][10]：
1. **列均匀性**：强化列特征以克服条纹噪声的列差异
2. **空间相关性**：增强关键区域的结构特征以解耦条纹噪声
3. **自相依性**：建立灵活的远程依赖关系，聚合上下文信息以微调全局均匀性

## 2. 模块工作原理和核心思想

### 整体架构
CSSC模块由三个并行分支组成，最终通过融合机制产生输出 [9][10]：

**数学表达式**：
```
Co = f1×1([SA, CA]) ⊙ SC + X
```
其中：
- `Co`：CSSC的输出特征
- `SA`：空间注意力分支的输出
- `CA`：列注意力分支的输出  
- `SC`：自校准分支的输出
- `⊙`：广播哈达玛积（逐元素乘法）
- `X`：输入特征

### 三大核心分支

#### **(1) 空间注意力分支（SAB）**
**目的**：增强空间相关性，突出关键区域的结构特征 [10][11]

**工作流程**：
- 对输入特征`X ∈ R^(C×H×W)`沿通道维度进行全局平均池化和最大池化
- 将两种池化响应拼接后通过3×3卷积和sigmoid激活获得空间权重
- 将空间权重与输入特征相乘生成输出`SA`

**数学表达式** [10]：
```
SA = f_s^(3×3)([f_Avg^spatial(X), f_Max^spatial(X)]) ⊙ X
```

#### **(2) 列注意力分支（CAB）**
**核心思想**：由于同一列的像素共享相同的放大器，同列特征应被赋予相似的校正系数以强调列依赖性 [11]

**创新设计**：
- **双列池化**：使用列平均池化和列最大池化（池化核大小为(H,1)）捕捉每列的不同语义响应 [11]
- **列权重机制**：将池化响应沿通道维度拼接，通过通道注意力机制生成列权重
- **双重校正**：分别对平均和最大池化结果应用注意力模型，实现更精细的列特征调制

**数学表达式** [11]：
```
Mc = [f_Avg^column(X), f_Max^column(X)]
Xa, Xm = Chunk2(CBL(Mc))
CA = X ⊙ F_Avg^column(Xa) ⊙ F_Max^column(Xm)
```

**与传统CCM的对比** [18]：
- 引入双列池化策略（相比CCM的单一池化）
- 增加列权重机制
- 采用双重校正策略，性能提升0.19 dB

#### **(3) 自校准分支（SCB）**
**目的**：实现强大的上下文信息建模，在每个空间位置建立长程依赖关系 [11]

**工作原理** [11]：
```
SC = δ_s(X + B2(conv(A2(X))))
```
其中：
- `A2(·)`：2×2平均池化（步长为2）
- `B2(·)`：双线性插值放大2倍
- 通过下采样-卷积-上采样的过程捕捉多尺度上下文信息

**优势**：灵活聚合列的全局上下文信息，微调不同特征层之间的全局均匀性 [10][11]

### 融合机制
三个分支的输出通过以下方式融合 [9]：
1. 将SA和CA拼接后通过1×1卷积降维
2. 与SC进行逐元素乘法
3. 添加残差连接（输入X）保持信息流

## 3. 总结

CSSC模块通过**三分支协同机制**有效解决了红外图像去条纹任务中的全局列特征建模问题：

**关键贡献**：
1. **列注意力分支**：通过双列池化和双重校正策略，精确捕捉列间的非均匀性差异
2. **空间注意力分支**：增强关键区域的结构表征，帮助区分条纹噪声和垂直背景细节
3. **自校准分支**：建立长程依赖关系，实现全局上下文的灵活聚合

**实验验证** [18]：
- 相比不使用空间注意力，PSNR提升0.83 dB
- 相比不使用自校准，PSNR提升0.34 dB  
- 相比传统CCM模块，PSNR提升0.30 dB

CSSC模块被嵌套在密集连接残差（DCR）结构中，通过特征重用和信息流增强，成为ASCNet实现优异去条纹性能的核心组件 [10]。
