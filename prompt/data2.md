【评价指标：WDA（加权域准确率）】
• 摒弃全局mAP，采用域级平均
• 公式: WDA = (1/D) × Σ(Domain_Accuracy_d)
• 核心思想: 强制模型在所有域上均衡表现
• 苏丹30张图 = 法国2130张图（等权重）

【关键洞察 - 创新启发点】

1. 验证集vs测试集性能鸿沟:
   • Val集: DEIM达到AP=0.504，性能优秀
   • Test集: DEIM仅0.318，下降37%
   → 说明: 模型严重过拟合到训练域分布

2. 小目标检测薄弱:
   • 所有方法AP_s < 0.1（大多<0.09）
   • Test集16.6%为小目标，但模型表现极差
   → 创新方向: 多尺度特征增强、注意力机制

3. 密集场景漏检:
   • AR_100普遍<0.4，说明高密度区域召回率低
   • UQ_8（117.9/图）等极端密集域表现堪忧
   → 创新方向: NMS策略改进、重叠目标处理

4. 域偏移脆弱性:
   • 苏丹、墨西哥等OOD域性能崩溃
   • 背景纹理与目标混淆（浅色土壤vs成熟麦穗）
   → 创新方向: 域自适应、对比学习、数据增强

5. 定位精度不足:
   • AP_75急剧下降（0.318→0.242→0.135）
   • 说明IoU>0.75的高质量检测框比例低
   → 创新方向: 边界框回归优化、Deformable机制
